# パーツ分解てとら
## 概要
文字などをパーツごとに分解することができる映像エフェクトです。  

<img width="1280" height="720" alt="image" src="https://github.com/user-attachments/assets/0a475ceb-1745-49c9-9ed0-c71879473e98" />

## 導入方法
### [最新バージョンをダウンロード](../../releases/latest)
ダウンロードしたファイルをダブルクリックして、表示される画面にしたがってインストールしてください。
> [!NOTE]
> ファイルをダブルクリックしてもインストールできない場合は、YMM4を起動して`ヘルプ(H)`>`YMM4用拡張子の関連付け`>`登録する`から拡張子の関連付けを行ってください。
> 再度ファイルをダブルクリックすることでインストール画面が表示されます。

## 使用方法
映像エフェクト「パーツ分解てとら」を使用してください。  
`パーツ分解てとら / 映像エフェクト`の中に入れた映像エフェクトが分解画像に適用されます。  
### 注意点
* `パーツ分解てとら / 映像エフェクト`の先頭に中心位置(水平位置:中央 垂直位置:中央 位置を保持:オン)を入れることをおすすめします。

  <img width="380" height="383" alt="image" src="https://github.com/user-attachments/assets/34787fea-6af6-4718-9a41-2195cbe0083d" />
  
* ランダム系エフェクトを使っているときやアニメーションにランダムを使っているときにエフェクトのパラメーターを操作すると描画がおかしくなることがありますが、プレビューや動画出力に影響はありません。  
* 描画に異変があるときはテキストなら「文字ごとに分割」をオンにすると直るかもしれません。

## アンインストール方法
1. YMM4を起動して`ヘルプ(H)`>`その他`>`プラグインフォルダを開く`をクリックする。
2. YMM4を終了する。
3. `PartsSplitTetra`という名前のフォルダを削除する。
